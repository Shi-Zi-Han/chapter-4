# 📊 交互式数据可视化系统

## 项目概述

这是一个基于Python数据可视化第4章的交互式图表展示系统，提供了完整的图表样式设置和实时预览功能。系统集成了五个不同类型的图表，支持用户自定义修改数据、颜色、样式和标记等参数。

## 🌟 功能特性

### 核心功能
- **五种图表类型**：图书采购柱状图、汇率走势折线图、产品销售分析、温度变化趋势、科赫雪花分形图
- **实时交互预览**：所有修改即时生效，支持防抖优化
- **主题切换**：支持浅色/深色主题，现代化UI设计
- **数据标记功能**：支持多种标记类型（圆形、方形、菱形、三角形、星形等）
- **导出功能**：支持PNG、SVG、JPG格式下载

### 交互控制
- **数据编辑**：实时修改图表数据值
- **颜色配置**：自定义系列颜色和样式
- **图表设置**：调整标题、线条样式等参数
- **响应式设计**：适配桌面和移动设备

## 📋 图表说明

### 1. 📚 图书采购情况分析
**目的**：展示两个地区对五类图书的采购数量对比
**数据来源**：模拟的图书采购记录
**关键发现**：小说类图书在两地采购量均最高，科技类图书采购量也相对较高，反映不同地区阅读偏好差异

### 2. 💱 美元/人民币汇率走势
**目的**：对比2017年7月和2019年7月美元兑人民币汇率变化
**数据来源**：历史外汇市场记录
**关键发现**：2017年汇率相对稳定（6.78-6.81），2019年波动较大（6.86-6.89），显示汇率受宏观经济因素影响明显

### 3. 📈 季度产品销售分析
**目的**：展示三种产品在四个季度的销售额变化趋势
**数据来源**：模拟真实销售场景
**关键发现**：产品A增长迅速，产品B稳步增长，产品C基数虽小但增幅明显，显示季节性消费特征

### 4. 🌡️ 15天温度变化趋势
**目的**：展示未来15天最高和最低气温的变化情况
**数据来源**：气象预报模拟
**关键发现**：温度波动明显，第11-13天出现最低值，随后回升，温差变化显示气候变化的不确定性

### 5. ❄️ 科赫雪花分形图案
**目的**：展示科赫雪花分形几何图形的数学美
**数据来源**：迭代算法生成
**关键发现**：具有自相似性和无限周长特性，在自然界中广泛存在，有助于理解复杂系统的结构规律

## 🚀 快速开始

### 运行方式
1. 直接双击打开 `interactive_visualization_optimized.html` 文件
2. 或在浏览器中拖拽文件打开
3. 支持所有现代浏览器（Chrome、Firefox、Safari、Edge）

### 系统要求
- 现代Web浏览器
- 网络连接（用于加载外部库）
- 不需要安装任何额外软件

## 🎮 使用指南

### 基本操作
1. **选择图表**：点击顶部的图表按钮切换不同图表
2. **修改数据**：在数据编辑面板中修改数值
3. **调整颜色**：使用颜色选择器自定义系列颜色
4. **设置样式**：调整线条样式、标记类型等参数
5. **查看说明**：每个图表都有详细的说明文字

### 高级功能
- **主题切换**：右上角点击太阳/月亮图标切换主题
- **数据标记**：为折线图添加自定义标记
- **导出图表**：点击下载按钮保存图表
- **撤销修改**：支持撤销和重置功能

## 🛠️ 技术架构

## 🔧 自定义开发

### 添加新图表
1. 在 `ChartManager.initializeChartData()` 中添加新图表配置
2. 更新 `switchChart()` 函数和HTML按钮
3. 添加对应的图表说明

### 修改样式
- CSS变量在 `:root` 中定义，支持主题切换
- 响应式断点在 `@media` 查询中设置

### 扩展功能
- 可在现有架构基础上添加新的控制面板
- 支持更多图表类型和数据格式

## 📊 图表类型说明

| 图表类型 | 适用场景 | 特性 |
|---------|---------|------|
| 柱状图 | 分类数据比较 | 显示分类间差异，支持堆叠 |
| 折线图 | 时间序列趋势 | 展示变化趋势，支持多系列 |

## 🎯 最佳实践

### 数据输入
- 数值数据应为正数
- 时间序列数据按时间顺序排列
- 分类数据使用清晰标签

### 视觉效果
- 颜色对比度要足够
- 标记大小适中，避免重叠
- 图例清晰可读

### 性能考虑
- 大数据集使用分页或聚合
- 复杂图表启用缓存
- 移动设备使用简化版本

## 🔍 故障排除

### 常见问题
1. **图表不显示**：检查网络连接和浏览器控制台
2. **数据修改无效**：确认点击"确认修改"按钮
3. **导出失败**：检查浏览器下载权限

### 技术支持
- 查看浏览器控制台错误信息
- 确保使用现代浏览器版本
- 检查文件路径和权限
